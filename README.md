# StudyMate Backend

Java/Spring Boot 기반 스터디 모집 서비스 백엔드 프로젝트입니다.

## 프로젝트 소개

StudyMate는 스터디를 모집하고 싶은 사용자가 모집글을 작성하고, 다른 사용자가 해당 스터디에 신청하거나 댓글을 남길 수 있는 스터디 모집/관리 서비스입니다.

본 프로젝트는 회원 관리, 스터디 모집글 관리, 댓글, 스터디 신청 기능을 제공하는 REST API 서버 구현을 목표로 합니다.

## 프로젝트 목적

- Java/Spring Boot 기반 REST API 서버 구현
- MySQL을 활용한 데이터베이스 설계 및 연동
- 회원, 게시글, 댓글, 신청 기능을 포함한 기본 웹 서비스 구조 구현
- 인증/인가, 검색, 페이징, 예외 처리 등 백엔드 기본 기능 구현
- 신입 백엔드 개발자 포트폴리오로 활용 가능한 개인 프로젝트 완성

- 페이징 처리

### 댓글 기능

- 댓글 작성

## MVP 범위

1차 구현 목표는 아래 기능까지입니다.

- 회원가입
- 로그인
- 스터디 모집글 CRUD
- 댓글 CRUD
- 스터디 신청/취소
- 검색
- 페이징
- 기본 예외 처리

## DB 테이블 초안

- users: 회원 정보
- studies: 스터디 모집글 정보
- comments: 댓글 정보
- applications: 스터디 신청 정보

## 테이블 관계 초안

- 회원 1명은 여러 개의 스터디 모집글을 작성할 수 있습니다.
  - users 1 : N studies

- 회원 1명은 여러 개의 댓글을 작성할 수 있습니다.
  - users 1 : N comments

- 스터디 모집글 1개에는 여러 개의 댓글이 작성될 수 있습니다.
  - studies 1 : N comments

- 회원 1명은 여러 스터디에 신청할 수 있습니다.
  - users 1 : N applications

- 스터디 모집글 1개에는 여러 명의 사용자가 신청할 수 있습니다.
  - studies 1 : N applications
